#RTF

Frequently used template function for [revel](http://revel.github.io) framework

## Using
### 1. Import
Import in your revel project app/init.go file

````
import	_ "github.com/baiyuxiong/rtf"
````

### 2. Use in template file
````
{{now}}
````
